Admin Portal / Settings
Branch Capacity
Tick the bays this branch has available. Bay #1 to Bay #6 are shown by default, and you can add more if a branch grows later.
These names appear in the Assigned To dropdown for Part Day, 2 Day Fit and 1 Week Fit bookings.
Tick the part-day slots available for this branch. The booking form still chooses the exact time and automatically assigns Morning before 12pm or Afternoon from 12pm.
Tick which multi-day slots are available. Choose the start and end day for 2-day or full-week bookings. Untick all rows if the branch has no multi-day/week fit capacity.
Blocking / Public Holidays
One public holiday per line. Empty Part Day and Full Day slots on that date will be blocked automatically.
Authority Roles / PINs
Set a PIN for each authority level, then tick exactly what that PIN can do. Leave Viewer with no PIN for default view-only access. These shared role PINs remain available as a backup login path.
Individual Users / Personal PINs
Create a named user, choose their role and branch access, then use Copy Invite to send them a setup link. Personal PINs are not shown after setup; use Reset PIN if someone forgets theirs.
Workflow Actions & Colours
Add, remove or recolour workflow buttons. Use key onsite for the action that should count vehicles on-site. Use key collected and tick Clear Others for the final collected/completed action.
Status Colours
These use the same simple colour picker style as workflow actions. Add matching status names here for Public Holiday, Not Available, Sick Leave, Warehouse, or any new blocked status.
Vehicle Fields
These populate the Make suggestions in the booking form and Summary sheet. The booking form Make field can also be typed manually for obscure makes.
